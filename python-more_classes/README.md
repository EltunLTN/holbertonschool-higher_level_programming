# Python - More Classes

This directory contains Python exercises that extend object-oriented concepts, mainly through progressive `Rectangle` class implementations and one algorithmic backtracking challenge.

## Learning Objectives

- Strengthen OOP design with constructors, validation, and methods
- Use class attributes, static methods, and class methods
- Implement custom string and repr behavior
- Track object instances and compare objects by area
- Apply backtracking to solve constraint-based problems

## Files

- `0-rectangle.py`: Defines an empty `Rectangle` class
- `1-rectangle.py`: Adds width and height with validation
- `2-rectangle.py`: Adds `area` and `perimeter` methods
- `3-rectangle.py`: Adds string representation using `#`
- `4-rectangle.py`: Adds eval-friendly `repr` representation
- `5-rectangle.py`: Adds deletion message and instance cleanup behavior
- `6-rectangle.py`: Adds class attribute for customizable print symbol
- `7-rectangle.py`: Adds instance counting and improved representations
- `8-rectangle.py`: Adds static method to compare rectangles by area
- `9-rectangle.py`: Adds class method to create square rectangles
- `101-nqueens.py`: Solves the N-Queens problem using backtracking

## Requirements

- Python 3.x
- Files should be executable where required
- Follow Holberton style and project constraints

## Usage

Run any script with:

```bash
python3 <filename.py>
```

Example:

```bash
python3 101-nqueens.py 8
```
