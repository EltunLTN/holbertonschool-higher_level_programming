# Python - Data Structures

This directory contains beginner Python exercises focused on lists, tuples, and string manipulation.

## Learning Objectives

- Work with lists and tuples in Python
- Use indexing and slicing safely
- Write functions that return updated data without side effects when required
- Practice basic algorithmic thinking with simple data transformations

## Files

- `0-print_list_integer.py`: Prints all integers of a list, one per line
- `1-element_at.py`: Retrieves an element from a list at a specific index
- `2-replace_in_list.py`: Replaces an element in a list at a given index
- `3-print_reversed_list_integer.py`: Prints a list of integers in reverse order
- `4-new_in_list.py`: Replaces an element in a copy of a list
- `5-no_c.py`: Removes all `c` and `C` characters from a string
- `6-print_matrix_integer.py`: Prints a matrix of integers
- `7-add_tuple.py`: Adds two tuples
- `8-multiple_returns.py`: Returns the length of a string and its first character
- `9-max_integer.py`: Finds the biggest integer in a list
- `10-divisible_by_2.py`: Finds all multiples of 2 in a list
- `11-delete_at.py`: Deletes an item at a specific position in a list
- `12-switch.py`: Swaps the values of two variables

## Requirements

- Python 3.x
- Files should be executable where required

## Usage

Run any script with:

```bash
python3 <filename.py>
```

Example:

```bash
python3 0-print_list_integer.py
```
