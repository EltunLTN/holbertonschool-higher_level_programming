# Python - Classes

This directory contains Python exercises focused on object-oriented programming with classes, attributes, methods, and basic data modeling.

## Learning Objectives

- Create and use Python classes
- Work with private attributes and properties
- Validate data in setters and constructors
- Implement instance methods and string representations
- Build and traverse a simple singly linked list

## Files

- `0-square.py`: Defines an empty `Square` class
- `1-square.py`: Defines `Square` with a private `size` attribute
- `2-square.py`: Adds size validation (`int` and `>= 0`)
- `3-square.py`: Adds an `area` method to compute square area
- `4-square.py`: Adds property getters and setters for `size`
- `5-square.py`: Adds `my_print` to print a square with `#`
- `6-square.py`: Adds `position` to print square with offset
- `100-singly_linked_list.py`: Implements a sorted singly linked list
- `101-square.py`: Adds custom string representation for `Square`
- `102-square.py`: Adds rich comparison for squares based on area

## Requirements

- Python 3.x
- Files should be executable where required
- Follow Holberton style and project constraints

## Usage

Run any script with:

```bash
python3 <filename.py>
```

Example:

```bash
python3 3-square.py
```
