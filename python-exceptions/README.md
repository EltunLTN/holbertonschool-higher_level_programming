# Python - Exceptions

This directory contains Python exercises focused on exception handling, safe function execution, and robust error management.

## Learning Objectives

- Understand how exceptions work in Python
- Use try and except blocks to prevent crashes
- Handle specific exception types cleanly
- Write helper functions that fail safely
- Raise custom exceptions when needed

## Files

- `0-safe_print_list.py`: Prints x elements of a list safely
- `1-safe_print_integer.py`: Prints an integer safely
- `2-safe_print_list_integers.py`: Prints only integer elements from a list safely
- `3-safe_print_division.py`: Divides two integers and handles division errors
- `4-list_division.py`: Divides list elements pairwise with full exception handling
- `6-raise_exception_msg.py`: Raises a NameError with a custom message
- `100-safe_print_integer_err.py`: Prints an integer and writes errors to stderr
- `101-safe_function.py`: Executes a function safely and handles exceptions
- `102-magic_calculation.py`: Reproduces specific bytecode logic using exceptions

## Requirements

- Python 3.x
- Files should be executable where required
- Code should follow task constraints from Holberton project instructions

## Usage

Run any script with:

```bash
python3 <filename.py>
```

Example:

```bash
python3 0-safe_print_list.py
```
